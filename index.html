Site Web Art pour Vivre

Ce site a été conçu en HTML et CSS dans le cadre d’un projet universitaire réalisé en équipe de trois. L’objectif était de développer un site vitrine pour une entreprise fictive du secteur événementiel. Nous avons imaginé “Art pour Vivre”, une société spécialisée dans l’organisation d’expositions artistiques à travers la France.

Jeu Bomberman en JavaFX

Ce projet, développé en JavaFX, consistait à reproduire le célèbre jeu Bomberman selon un cahier des charges précis. Le programme intègre différentes fonctionnalités : gestion des collisions, inventaire des bombes, système de points de vie et interface dynamique.
Voir le dépôt GitHub

Application L’Ô Rhône en JavaFX

Réalisée en binôme, cette application JavaFX permet la gestion des données d’une station de traitement d’eau. Elle offre la possibilité de gérer les clients, leurs commandes et les informations associées. Ce projet m’a permis de renforcer mes compétences en conception d’interfaces et en structuration de données.
Voir le dépôt GitHub

Jeu 2048 en JavaFX

Ce projet individuel visait à recréer le jeu 2048 avec JavaFX. J’ai conçu une interface claire et intuitive permettant à l’utilisateur de jouer facilement, tout en appliquant les mécaniques de fusion et de déplacement des tuiles.
Voir le dépôt GitHub

Jeu Lights Off en JavaFX

Ce jeu a été développé individuellement en JavaFX. L’objectif était de concevoir une version interactive du puzzle “Lights Off”. L’application propose une interface simple mais efficace, permettant de tester sa logique en essayant d’éteindre toutes les lumières. Voir le dépôt GitHub
